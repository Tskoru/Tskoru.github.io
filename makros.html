Макросы

Что такое макросы?

Макрос - записанная последовательность команд, которую надо записать один раз, а потом можно поручить выполнить excel.

Первый макрос, который я сделала - макрос вставки. Он помогает вставлять любую фразу на любую ячейку простым нажатием на комбинацию клавиш ctrl и любой другой на ваш выбор. Мой макрос вставки работает только на одном листе блягодаря специальному коду(If Worksheets(ActiveSheet.Name).Name = "Лист1" Then (МАКРОС) End if):

Sub Макрос3() ' ' Макрос3 Макрос ' ' Сочетание клавиш: Ctrl+й ' If Worksheets(ActiveSheet.Name).Name = "Лист1" Then ActiveCell.FormulaR1C1 = "catdog" Range("A2").Select End If End Sub

Следующий макрос может находить определённые буквы, или сочетания букв, или сочетания цифр и заменять их на другие. Мой макрос менял слово catdog на dogcat:

Sub Макрос4() ' ' Макрос4 Макрос ' ' Сочетание клавиш: Ctrl+ц ' Cells.Find(What:="catdog", After:=ActiveCell, LookIn:=xlFormulas, LookAt _ :=xlPart, SearchOrder:=xlByRows, SearchDirection:=xlNext, MatchCase:= _ False, SearchFormat:=False).Activate Cells.Replace What:="catdog", Replacement:="dogcat", LookAt:=xlPart, _ SearchOrder:=xlByRows, MatchCase:=False, SearchFormat:=False, _ ReplaceFormat:=False End Sub

Последний макрос, который я сделала, менял толщину шрифта и делал надпись курсивной:

Sub Макрос5() ' ' Макрос5 Макрос ' ' Сочетание клавиш: Ctrl+у ' Selection.Font.Bold = True Selection.Font.Italic = True End Sub
